# Рабочая документация проекта «Комиссионка»

Каталог содержит детальную документацию по каждому модулю системы. Требования к документации: структура с оглавлением, версия в заголовке, стек и архитектура, функциональные блоки со ссылками на код, модель данных и доступ к Prisma.

## Документы

| Модуль | Файл | Описание |
|--------|------|----------|
| Web-приложение | [app-komissionka.md](app-komissionka.md) | Next.js-приложение, API, админка, аутентификация, Deploy |
| Агент к модели ИИ | [agent.md](agent.md) | HTTP/CLI агент, LLM, инструменты, кэш, Deploy |
| Телеграм-бот | [telegram-bot.md](telegram-bot.md) | Long polling, привязка профилей, вызов агента, Deploy |

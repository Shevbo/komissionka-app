# ClipAngel: эмуляция ввода в веб-консоль

ClipAngel — менеджер буфера обмена. В нём есть **эмуляция набора с клавиатуры**, которая помогает, когда copy-paste в веб-консоль (VPS, терминал) не работает.

---

## 1. Скачивание и установка

1. Откройте: https://sourceforge.net/projects/clip-angel/
2. Нажмите **Download** (зелёная кнопка).
3. Распакуйте архив и запустите `ClipAngel.exe`.

**Требования:** Windows Vista+, .NET Framework 4.52+ (обычно уже установлен в Windows 10/11).

---

## 2. Настройка горячей клавиши

1. Кликните правой кнопкой по иконке ClipAngel в трее → **Settings** (Настройки).
2. Найдите пункт **«Hotkey to simulate fast keyboard input»** / **«Горячая клавиша для эмуляции быстрого ввода»**.
3. Нажмите в поле и задайте сочетание, например **Ctrl+Shift+V**.
4. Нажмите **OK**.

---

## 3. Использование (пример: SSH-ключ в веб-консоль)

1. Скопируйте текст (например, SSH-ключ):
   ```powershell
   Get-Content $env:USERPROFILE\.ssh\id_ed25519.pub | Set-Clipboard
   ```
   или откройте файл и Ctrl+C.

2. Переключитесь на окно веб-консоли (щёлкните по нему).

3. Нажмите **Ctrl+Shift+V** (или своё сочетание). Текст будет «напечатан» как с клавиатуры.

4. При необходимости добавьте Enter и введите следующую команду:
   ```
   chmod 600 ~/.ssh/authorized_keys
   ```

---

## Примечания

- Работает с любым текстом — SSH-ключи, команды, пароли.
- Эмуляция ввода может работать там, где Ctrl+V не срабатывает.
- ClipAngel по умолчанию запускается с Windows (опция в настройках).
